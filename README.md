BullyStop é um sistema web de denúncia de bullying escolar para permitir que alunos, professores, e qualquer funcionário de um instituto escolar faça denúncias sobre qualquer ato de agressão. 

O objetivo principal desse projecto é impedir com que qualquer indivíduo que frequenta um ambiente escolar sinta-se excluído ou ameaçado dentro da instituição fre
quentada por ele.
